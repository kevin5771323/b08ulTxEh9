# 项目简介

## 核心功能
- 基于YOLOv8的杂草分类检测系统，支持12类杂草识别
- 提供完整源码、14000张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 提供界面选择显示标签和原图，可选择检测模型
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 包含完整的项目运行教程

## 适用场景
- 农业杂草识别与管理
- 农田监测与作物保护
- 农业自动化研究
- 计算机视觉教学与实验

## 技术特点
- 采用PyCharm+Anaconda或VSCode+Anaconda开发环境
- 依赖Python 3.9、OpenCV-Python、PyQt5、Ultralytics、PyTorch 1.9等技术栈
- 提供完整的UI界面源文件和图标资源
- 已配置成功环境，可直接使用
